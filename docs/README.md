# OpenGlide Decision Map

Open-source decision support maps for glider pilots.

The goal of this project is to help pilots make safer decisions by combining terrain, local flying knowledge, airfields, outlanding options and configurable safety margins into personalized glide decision maps.

> If a pilot can correctly interpret the map in turbulent air within three seconds, we have achieved the goal.

## First dataset

The first regional dataset is:

- Switzerland
- Schänis LSZX edition

# Mapping Guidelines

## Purpose

The map must help a pilot make a safe decision quickly.

If an element does not support this goal, it should not be shown on the cockpit map.

## Decision point

A decision point is a location where a pilot may reasonably choose between two or more safe strategies.

Examples:

- Walenstadt
- Weesen
- Sargans
- Panixerpass
- Chrüzlipass

Not every village or landmark is a decision point.

## Corridor

A corridor is a preferred route connecting decision points while minimizing pilot workload and maximizing safety.

A corridor may follow:

- valley
- pass
- ridge
- terrain transition
- flatland route

## Outlanding rating

Initial rating system:

| Code | Meaning |
|---|---|
| P | Preferred |
| S | Suitable |
| E | Emergency only |

The rating is not only a geometric quality. It reflects local knowledge and operational usefulness.

## Paper map principle

The printed map should be readable in turbulent air within three seconds.

This means:

- simple symbols
- limited text
- strong hierarchy
- no unnecessary information
- no calculation clutter

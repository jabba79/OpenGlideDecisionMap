# Vision

## Why

Every glider pilot knows the feeling.

You are returning from a cross-country flight.  
The weather changes.  
Cloudbase gets lower.  
Home still seems within reach.

The hardest part is often not flying.

The hardest part is making the right decision.

OpenGlide Decision Map helps pilots make safer decisions by combining terrain, local flying knowledge, airfields, outlanding options and configurable safety margins into personalized decision maps.

OGDM is not a navigation system.  
It is a decision support system.

## Mission

Create open, configurable and printable decision maps that help glider pilots make safer decisions.

## Long-term vision

Build an open framework where local gliding communities can contribute their knowledge to create personalized decision maps for mountain flying regions worldwide.

## First milestone

Switzerland — Schänis LSZX edition.

## Design goal

If a pilot can correctly interpret the map in turbulent air within three seconds, we have achieved the goal.

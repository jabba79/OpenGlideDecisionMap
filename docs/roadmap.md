# Roadmap

## Version 0.1 — Schänis

Goal: create the first usable decision map for LSZX Schänis.

Scope:

- project architecture
- data model
- QGIS project
- Schänis home airfield profile
- first airfields
- first outlanding sites
- first decision corridors
- first printable map

No advanced automation yet.

## Version 0.2 — Eastern Switzerland

- extend dataset around eastern Switzerland
- add more corridors
- add more decision points
- improve outlanding site classification

## Version 0.3 — Switzerland

- broader Swiss dataset
- more airfields
- region-based profiles
- improved print layouts

## Version 0.5 — Personal profiles

- pilot profiles
- aircraft profiles
- configurable glide ratios
- configurable arrival margins

## Version 1.0 — Framework

- stable data model
- processing engine
- printable maps
- QGIS workflow
- documentation for community contributions

## Future ideas

- tablet output
- flight cards
- mission packs
- weather context
- NOTAM context
- additional regional datasets such as Austria or French Alps

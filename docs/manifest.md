# OpenGlide Decision Map Manifest

Flying in the mountains is not about finding the shortest path.

It is about making good decisions.

Good decisions come from experience.

OGDM preserves this experience.

It transforms local flying knowledge into maps that help pilots make safer decisions.

The map should never replace judgement.

The map should support judgement.

Every local gliding club has knowledge.

Every experienced pilot has knowledge.

We believe this knowledge should not disappear.

It should be shared.

Openly.

For the next generation of pilots.

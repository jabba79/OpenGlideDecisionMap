# Contributing

OpenGlide Decision Map welcomes contributions from pilots, instructors, clubs and developers.

## Contribution types

You can contribute:

- local flying knowledge
- outlanding site information
- decision points
- corridors
- QGIS styles
- Python code
- documentation
- testing feedback

## Flight knowledge contributions

When contributing flying knowledge, please include:

- location
- description
- source
- date of last review
- confidence level
- whether the information may be published publicly

## Safety note

OGDM is a decision support project.  
It does not replace official aviation publications, pilot judgement, training or legal responsibility.

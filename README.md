# OpenGlide Decision Map

> Open-source decision support maps for glider pilots.

OpenGlide Decision Map (OGDM) is not a navigation system.  
It is a decision support system built by glider pilots, for glider pilots.

The goal is to help pilots quickly understand their safe options: home airfield, alternate airfield, or outlanding site.

> If a pilot can correctly interpret the map in turbulent air within three seconds, we have achieved our goal.

## First dataset

The first regional dataset is:

- Switzerland
- Schänis LSZX edition

## Project status

Early concept, data model and QGIS prototype phase.

## Core idea

Same data, different outputs:

- printable paper map
- QGIS project
- tablet map
- later: mission cards and flight packs
